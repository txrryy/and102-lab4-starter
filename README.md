Campground Explorer
Campground Explorer is an Android application that allows users to browse and view details about various national park campgrounds. The app connects to the National Park Service (NPS) API to fetch real-time data, including descriptions, locations, and high-quality imagery.

Completed Features
The following required functionality is completed:

[x] RecyclerView Integration: Displays a list of campgrounds fetched from the NPS API.

[x] Custom Data Modeling: Uses Kotlin Serialization to parse nested JSON data into structured objects.

[x] Image Loading: Implements the Glide library to asynchronously download and cache campground images.

[x] Intent-Based Navigation: Users can tap on a campground to navigate from the main list to a detailed view.

[x] Detail Screen: Displays specific campground information, including the name, a full description, and latitude/longitude coordinates.

[x] Secure API Handling: Uses an apikey.properties file to manage sensitive API keys securely without hard-coding them.

The following stretch features are implemented:

[x] Styled UI: View elements are styled within .xml files to enhance the visual presentation of the campground list and details.
